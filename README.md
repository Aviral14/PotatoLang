# PotatoLang

Lexer and SLR Parser for PotatoLang Programming Language

# Guides

To build the project, execute -

```
$ make all
```

To run the project, execute -

```
$ build/compiler <filename>
```

To delete all the build files, execute -

```
$ make clean
```
